Hello World

We will start with a very simple actor, HelloWorldActor: 

import org.agilewiki.jactor2.core.*;
import org.agilewiki.jactor2.core.threading.*;
import org.agilewiki.jactor2.core.messaging.*;
import org.agilewiki.jactor2.core.processing.*;

public class HelloWorldActor extends ActorBase {

    public HelloWorldActor(final MessageProcessor _messageProcessor) throws Exception {
        initialize(_messageProcessor);
    }
    
    public Request<String> getGreetingReq() {
        return new Request<String>(getMessageProcessor()) {
            @Override
            public void processRequest(final Transport<String> _transport) throws Exception {
                _transport.processResponse("Hello world!");
            }
        };
    }
}

There is a lot here to discuss, but we should first note that this actor is completely unjustifiable, except that is is simple enough to use as a first example.

ActorBase

For HelloWorldActor to be an actor, it must implement Actor so that the getMessageProcessor can be used to access the actor's message processor. The easy way to do this is to extend ActorBase.

MessageProcessor

A message processor is a light weight thread that processes messages for one or more actors. An actor can not function without one. So in the constructor of HelloWorldActor, the method ActorBase.initialize is used to assign the actor's message processor.

Request<RESPONSE_TYPE>

A Request is a factory for RequstMessage. Request has an abstract method, processRequest, which is called on the thread of the actor which implements the subclass of Request. The HelloWorldActor.getGreetingReq method creates a request that simply returns the string "Hello world!".

Transport<RESPONSE_TYPE>

Transport is used to return and process a result on the thread of the actor which created the RequestMessage.

Here then is a program that exercises HelloWorldActor, HelloWorld: 

import org.agilewiki.jactor2.core.*;
import org.agilewiki.jactor2.core.threading.*;
import org.agilewiki.jactor2.core.messaging.*;
import org.agilewiki.jactor2.core.processing.*;

public class HelloWorld {
    public static void main(final String[] _args) throws Exception {
        ModuleContext context = new ModuleContext();
        try {
            NonBlockingMessageProcessor messageProcessor = new NonBlockingMessageProcessor(context);
            HelloWorldActor helloWorldActor = new HelloWorldActor(messageProcessor);
            Request getGreetingReq = helloWorldActor.getGreetingReq();
            String response = getGreetingReq.call();
            System.out.println(response);
        } finally {
            context.close();
        }
    }
}
