Getting Started

You will need three things to work with JActor2 core:

  1. JDK 7 or later.
  2. The JActor2 core jar file. And
  3. A test script. For windows, you can use test.bat: 
    del *.class
javac *.java
java -classpath jactor2-core-0.0.2.jar;. %1
