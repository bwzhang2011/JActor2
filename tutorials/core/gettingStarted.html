Getting Started

You will need four things to work with JActor2 core:

  1. JDK 7 or later.
  2. slf4j.
  3. The JActor2 core jar file. And
  4. A test script. For windows, you can use test.bat: 
    del *.class
javac -classpath jactor2-core-0.0.3.jar;slf4j-api-1.7.5.jar *.java
java -classpath jactor2-core-0.0.3.jar;slf4j-api-1.7.5.jar;slf4j-simple-1.7.5.jar;. %1
    Othrwise, you can likely use test.sh: 
    test.sh is
rm -fr *.class
javac -cp jactor2-core-0.0.3.jar:slf4j-api-1.7.5.jar *.java
java -cp jactor2-core-0.0.3.jar:slf4j-api-1.7.5.jar:slf4j-simple-1.7.5.jar:. $1
